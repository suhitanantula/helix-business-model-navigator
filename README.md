## The Double-AAA Framework

A 3x3 matrix positioning companies based on:

**Vertical Axis: Customer Action (AAA Progression)**
- ASSIST: AI supports customers
- AUGMENT: AI co-creates with customers
- ADAPT: AI acts autonomously for customers

**Horizontal Axis: Execution (AAA Progression)**
- ASSIST: Structured, rule-based execution
- AUGMENT: Adaptive, learning-based execution
- ADAPT: Autonomous, AI-orchestrated execution

### The 9 Quadrants

| | Assist Execution | Augment Execution | Adapt Execution |
|---|---|---|---|
| **Adapt Customer** | Q7: Traditional Proxy | Q8: Intelligent Proxy | Q9: Autonomous Orchestrator |
| **Augment Customer** | Q4: Co-Creation Assistant | Q5: Dynamic Curator | Q6: Adaptive Partner |
| **Assist Customer** | Q1: Basic Assistant | Q2: Smart Assistant | Q3: Intelligent Assistant |
